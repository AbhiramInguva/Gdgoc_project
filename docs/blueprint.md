# **App Name**: PhishAware

## Core Features:

- Data Upload and Processing: Accept user uploaded data (email, phishing reports, text) and process it for analysis.
- Risk Analysis: Analyze uploaded data to determine the risk level (Low, Medium, High) based on detected phishing indicators.
- Suspicious Content Detection: Use AI to identify and highlight suspicious words and sender mismatches within the uploaded data. The tool will evaluate context and likelihood to classify indicators and calculate a risk score
- Data Visualization Panel: Display uploaded data with highlighted text, icons (⚠️, ❌, ✅), risk level indicators, and color-coded elements (green, yellow, red).
- Awareness Animation Panel: Display educational animations based on uploaded data.  Animations explain potential issues and user takeaways to prevent phishing attacks.
- Phishing Education Section: Provide a scroll-down section with information on various phishing attack types (email, link spoofing, urgent action scams), including identification tips.
- Real-World Examples: Show example phishing messages with highlighted dangerous elements and explanations.

## Style Guidelines:

- Primary color: Deep sky blue (#32A2EF) for clarity and trust.
- Background color: Light gray (#F0F4F7) for a clean and modern look.
- Accent color: Coral (#FF7F50) to draw attention to key interactive elements and risk indicators.
- Body and headline font: 'Inter', a sans-serif for readability and modern feel.
- Use clear and intuitive icons for warnings, success indicators, and types of phishing attacks.
- Implement a split-screen layout with the data visualization panel on the left and the awareness animation panel on the right.
- Use smooth and subtle animations like fades, slides, and highlights to guide the user's attention.
